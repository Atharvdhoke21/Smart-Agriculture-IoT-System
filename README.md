# 🌱 IoT-Based Smart Agriculture Monitoring System

## 📌 Overview

This project is an IoT-based smart agriculture system designed to monitor environmental conditions like **soil moisture, temperature, and humidity** and automate irrigation using a **NodeMCU (ESP8266)** and **Blynk app**.

---

## ⚙️ Features

* Real-time monitoring of soil moisture, temperature, and humidity
* Automatic irrigation using relay and water pump
* Remote monitoring via Blynk mobile app
* Notification alerts for system activity

---

## 🛠️ Components Used

* NodeMCU (ESP8266)
* Soil Moisture Sensor
* DHT11 Sensor
* Relay Module
* Water Pump
* LDR Sensor

---

## 🧠 Working Principle

Sensors collect environmental data and send it to the NodeMCU.
The controller processes data and:

* Activates pump when soil is dry
* Sends data to Blynk app for remote monitoring

---

## 📊 Results

* Real-time monitoring on mobile app
* Automated irrigation system
* Efficient water usage
